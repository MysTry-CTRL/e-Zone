# 🚀 e-Zone — Futuristic Ebook Platform

e-Zone is a modern, animated ebook web platform designed for selling and managing Bangla and English PDF books.  
It features a futuristic UI, admin-controlled uploads, category-based browsing, and a unified authentication system.

---

## ✨ Features

### 🎨 UI & Experience
- Light & Dark mode support
- Animated backgrounds on all major pages
- Smooth loading animation with delayed transition
- Playful animated shapes on homepage hero section
- Fully responsive design

---

### 🔐 Authentication System
- Universal control button for:
  - Login
  - Registration
  - Account access
- Secure admin-only credentials
- No guest browsing mode

---

### 👑 Admin Controls
Admin can:

- Upload books as PDF
- Upload cover files as PDF
- Select book category
- Mark/unmark books as featured
- Manage descriptions and book details

---

### 📚 Book System

#### Categories
- Love and Romance
- Fiction
- Non-fiction
- Science Fiction

#### Display Logic
- Homepage shows **3 featured books per category**
- Books page shows:
  - 3 featured books
  - 1 "Show All" category button

---

### 🛒 Purchase Flow
Instead of QR codes, each book displays:
- Step-by-step instructions for purchasing
